Tak
a beautiful game
Info:
Tak is an abstract board game similar to Chess and Go, created in 2016. This is a scrappy web implementation of Tak. I'm a beginner, so the graphics are lacklustre, you can play moves using Tak's PTN notation and only local multiplayer is available. There may be some minor or even major bugs. Nevertheless, I hope you enjoy it!
